‚ö°‚ö°‚ö° Description
This project provides a lightweight and flexible system that converts any input text‚ÄîThai, English, Japanese, or any language supported by your font‚Äîinto a fully-generated 3D model (.OBJ).
The pipeline uses a bitmap-based approach to ensure high compatibility without relying on heavy libraries or platform-specific text rendering engines.

Users simply input text, and the program outputs a clean, extruded 3D mesh that preserves the shape of the selected font.
This is ideal for creating 3D titles, logos, printable text objects, AR/VR assets, or integrating into AI systems that need text-to-3D capabilities.

‚≠ê‚≠ê‚≠ê Key Features

‚óè Convert any text into a 3D model (.OBJ)

‚óè Supports Thai, English, Japanese, and multi-language fonts

‚óè Bitmap ‚Üí Contour ‚Üí Polygon ‚Üí 3D Mesh pipeline

‚óè Automatic font scaling to ensure all characters fit

‚óè Height-normalized 3D output for consistent sizing

‚óè Works on any machine (no FreeType or platform dependencies)

‚óè Ready for integration with AI pipelines, web apps, or 3D editors
